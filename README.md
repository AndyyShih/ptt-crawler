# ptt-crawler

## 程式構想:<br>
首先設定欲爬文之版塊名稱，透過更改(line46)的range來決定要抓取幾頁的貼文<br>

## 迴圈的設定為:<br>
1. 點擊貼文(line49 to 55)<br>
2. 抓取貼文資料(line57 to 90)<br>
3. 上一頁(line92)<br>
4. 點擊下一篇貼文<br>
